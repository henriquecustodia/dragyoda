Drag-and-drop's module for AngularJS

## How to install 
`npm i && bower i`

## Quick start
```javascript
    angular.module('<yourModule>', ['hDragyoda']);
```


```html
    <div h-draggable>Draggable Element</div>
    <div h-droppable>Droppable Element</div>
``` 

To run the example,  just type the npm command:
```
    npm start
```

Enjoy!
